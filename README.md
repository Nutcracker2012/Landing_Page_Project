## Instructions
1, The project should have a clear structure. All files shown must be present and the app must successfully render a home page with clear design and functionality added when index.html is loaded in the browser. No errors should display in console.
2, All features are usable across modern desktop, tablet, and phone browsers.
3, Dynamicly add/remove active class to section within viewport & Styling has been added for active states.
4. There are at least 4 sections that have been added to the page & Navigation is built dynamically as an unordered list.
5. Create scroll event & It should be clear which section is being viewed while scrolling through the page.
6. When clicking an item from the navigation menu, the link should scroll to the appropriate section.
